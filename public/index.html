Demo Talking Points

What to say during your presentation

  • BFS guarantees the shortest path on this unweighted grid because it explores layer by layer.
  • DFS prioritizes depth first, so it can find a route quickly but not necessarily the shortest one.
  • A* blends the real cost so far with Manhattan distance to focus the search toward the goal.
  • All three algorithms use only four directions here: right, down, left, and up.
